# Dell OS6 System Health Zabbix Template

![Zabbix](https://img.shields.io/badge/Zabbix-Monitoring-red.svg)
![Dell](https://img.shields.io/badge/Dell-OS6-blue.svg)

A dedicated monitoring solution for Dell Networking OS6-series switches, focusing on critical hardware health metrics.

## 🚀 Key Features

*   **CPU & Memory:** Real-time utilization tracking for switch processors and RAM.
*   **Environmental Monitoring:** Temperature sensor tracking with custom threshold triggers.
*   **Hardware Status:** Health monitoring using proprietary Dell MIBs.

## 🛠️ Tech Stack

*   Zabbix
*   SNMP
*   Dell Networking MIBs
